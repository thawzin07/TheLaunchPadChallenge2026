# The LaunchPad Challenge 2026 - Folder Index

## One-Minute Context

The LaunchPad Challenge is a four-week open AI build sprint for students in Singapore. Teams of 1 to 3 build any serious AI project or research idea, submit by 31 July 2026 at 23:59 SGT, and are judged by domain-relevant professionals. The evaluation emphasizes problem understanding, justified approach, evidence, real-world constraints, and honest limitations.

## Core Submission Target

Required deliverables:

- Public or judge-accessible GitHub repository.
- Public YouTube demo video, maximum 3 minutes.
- Write-up, maximum 1,000 words, structured by the five judging pillars.
- Participant profile with introduction, short message to judges, and resume.

Judging pillars:

1. Problem
2. Approach
3. Evidence
4. Constraints
5. Honesty and Trajectory
